# GitHub Upload Checklist

Before making the repository public:

- [ ] Confirm the full senior design report is okay to share publicly.
- [ ] Confirm all selected images are okay to share publicly.
- [ ] Check that the Arduino code is the version you want shown.
- [ ] Add ORU Racing Instagram link if you want it included.
- [ ] Remove any private emails, phone numbers, or team-only notes before publishing.
- [ ] Make sure the README states that paddle shifting was your individual work and ignition-system work was a major team contribution you were heavily involved in.
- [ ] Make sure the README states that harness installation/integration was team-level work you contributed to.
- [ ] Make sure the status says the electrical system was about 90% complete, not fully vehicle-tested.

Suggested repo name:

```text
fsae-electrical-systems-portfolio
```

or

```text
fsae-paddle-shift-ignition-harness
```
